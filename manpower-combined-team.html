  • งานเฉพาะวลัยลักษณ์ + customization
  • ทดสอบ SIT/UAT/E2E + เอกสารส่งมอบทุกงวด (รวม M2 ของ Track B ด้วย)

= Phase 2 ของมหิดล (build ทีเดียวใช้ 2 ที่)
  • Dental Lab ★ (หัวใจ) + Mobile 3 แอป ★

✅ ทำไมดีกว่า: (1) ไม่ build ซ้ำ — มหิดลทำให้ทั้ง Phase 2 เขา + วลัยลักษณ์ · (2) สถาปัตยกรรมชุดเดียว → Integration ระหว่าง Track A/B ง่าย · (3) Track B เป็นคลินิกเฉพาะทาง (Dental Lab/Micro/Patho/CSSD/IC) → ความรู้ domain มหิดลตรงจุด · (4) greenfield → วิเคราะห์ไหลเข้า build ได้เลย ลดเวลา

🔶 กลไก Fallback:
  • Checkpoint 1 — ที่ M2 (8 พ.ย.): มหิดลต้อง commit แผน+เดดไลน์ Track B เป็นลายลักษณ์ ถ้ายังไม่ชัด → ยกระดับความเสี่ยง + เริ่มกัน Dev ทีมวลัยลักษณ์ไว้
  • Checkpoint 2 — กลาง ธ.ค.: ถ้ายังไม่เห็น increment Dental Lab → trigger fallback ดึง ซี/โอปอล์ มาลุย Dental Lab + Mobile หัวใจเอง
  • กันกำลังสำรอง: วาง Dev ทีมวลัยลักษณ์ในสาย Track A ให้มี slack ~15–20% ช่วง ธ.ค.–ม.ค. เพื่อ absorb หัวใจ Track B ได้ถ้าจำเป็น
  • ลำดับ descope: ถ้าคนไม่พอจริง → โมดูลรอง (CSSD/IC/KPI/PR/MIS) เจรจาส่งเป็นเฟสถัดไปก่อน · Dental Lab + Mobile ไม่ descope
⚠️ ระวังกำลังคนทีมวลัยลักษณ์: ถ้า fallback ทำงานเต็ม (ทีมวลัยลักษณ์รับ Track A + Integration + Migration + Dental Lab + Mobile) → Dev 3 คน เกินกำลังหนักทันที เหมือนแผน 7 คนเดิม → fallback จึงต้องคู่กับ “สิทธิ์เพิ่มคน/ยืม Dev มหิดลเพิ่ม” ในกรณีฉุกเฉิน

6สิ่งที่ต้องเคาะ/เจรจา

  1. ล็อกเดดไลน์ Track B เข้า milestone เรา (MOU/SLA back-to-back) — มหิดล commit วันส่งรายโมดูล โดยเฉพาะ Dental Lab + Mobile ให้ทันมา integrate ก่อน M3 (18 มี.ค.) · กำหนด drop-dead + checkpoint ตามข้อ 5
  2. เอกสาร M2 ของ Track B ยังต้องส่ง — แม้ build แบบ agile (วิเคราะห์→Dev เลย) แต่ TOR/งวด M2 ต้องมี SRS/DB schema/Architecture ของ Track B → ชมพู่ รับ design จากมหิดลมาเรียบเรียงเป็นเอกสารส่งมอบ
  3. Source code ownership + warranty 2 ปี ของ Track B — มหิดลอาจถือเป็น product Phase 2 ตัวเอง → ต้องเคลียร์สิทธิ์ส่งมอบ source ให้วลัยลักษณ์ + ใครรับประกัน Track B 2 ปี
  4. เคลียร์คน straddle 2 stream — ก้อง/โอ (SA) + หมี (DB) ต้องแบ่งเวลา Track A (ประกบทีมวลัยลักษณ์) vs Track B (มหิดล) → ตกลงสัดส่วนชัด ไม่ให้แย่งกันช่วง ก.ย.–ต.ค.
  5. ล็อกคนคอขวดสาย Track A: พีช (Integration, เหลือ ~0.6) + จีเน่ (Migration, ~1.6) ดึงยากแต่จำเป็น → เจรจาก่อน + หา backup
