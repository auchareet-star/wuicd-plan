1ขอบเขตการทดสอบเชื่อมต่อ (Integration Scope)

การทดสอบ SIT ครอบคลุม Interface การแลกเปลี่ยนข้อมูลระหว่าง DIS กับระบบภายนอกดังนี้

ระบบปลายทางรูปแบบเชื่อมต่อข้อมูลที่แลกเปลี่ยน
HAIT Plus (HIS เดิม)REST API / HL7ข้อมูลผู้ป่วย (HN, สิทธิ, แพ้ยา), ส่งรายการรักษากลับ
SAPAPI / Interface fileรายการเรียกเก็บเงิน, บัญชีลูกหนี้, GL
PACSDICOM / WADOภาพรังสีทันตกรรม, worklist
LIS/RISHL7 / APIคำสั่งตรวจ Lab, ผลตรวจ
⚠️ ข้อกำหนด: ต้องจองคิว "ทีม HIS เดิม" ล่วงหน้าเพื่อทดสอบ HAIT Plus บนสภาพแวดล้อมทดสอบ (Test/Staging) ที่มีข้อมูลตัวอย่างที่ผ่านการ mask แล้ว
