1สภาพแวดล้อมระบบ (Environments)

แยกสภาพแวดล้อมเพื่อความปลอดภัยและคุณภาพ — พัฒนา/ทดสอบไม่กระทบ Production

สภาพแวดล้อมวัตถุประสงค์สเปก/ที่ตั้ง (กรอก)
Developmentพัฒนา/Config โดยทีม Dev

2สเปกเซิร์ฟเวอร์และครุภัณฑ์

สเปกครุภัณฑ์ตามสัญญาอยู่ใน ภาคผนวก ค — ตารางนี้สรุปองค์ประกอบหลักสำหรับการติดตั้ง

องค์ประกอบจำนวนสเปก/หมายเหตุ
Application Server CPU/RAM/Disk ตามภาคผนวก ค
Database Server HA/สำรองข้อมูล
Web / Reverse Proxy SSL/Load balance
Storage / Backup สำหรับภาพรังสี/PACS index
อุปกรณ์เครือข่าย/ความปลอดภัย Firewall/VPN

3ขั้นตอนการติดตั้ง (Installation Checklist)

  • ตรวจรับครุภัณฑ์/เซิร์ฟเวอร์ตามภาคผนวก ค
  • ติดตั้งระบบปฏิบัติการ + patch ความปลอดภัย
  • ติดตั้งฐานข้อมูลและ middleware ที่จำเป็น
  • Deploy แอปพลิเคชัน DIS (Web) แต่ละ environment
  • ตั้งค่า config การเชื่อมต่อ HAIT Plus, SAP, PACS, LIS/RIS, WU-HRMS
  • ตั้งค่า Backup/Restore และทดสอบกู้คืน
  • ทดสอบ Smoke Test หลังติดตั้งทุก environment
  • บันทึกค่า config และส่งมอบ Installation Guide

4เครือข่ายและความปลอดภัย

  • การเข้ารหัสการสื่อสาร (TLS/SSL) ทุกช่องทาง
  • การแบ่ง network zone และกฎ Firewall สำหรับ interface ระบบเดิม
  • การยืนยันตัวตน/สิทธิ์ (Authentication & RBAC) เชื่อม WU-HRMS
  • การจัดเก็บ Log / Audit Trail ตามมาตรฐาน HA
  • การสำรองข้อมูลและแผนกู้คืน (Backup & DR)
⚠️ ข้อควรระวัง: การเปิด port/firewall เพื่อเชื่อม HAIT Plus ต้องประสานทีม HIS เดิมและฝ่าย IT มหาวิทยาลัยล่วงหน้า
