4คำแนะนำ Build / Deploy

สรุปขั้นตอนสำหรับให้ทีม รพ. สร้างและติดตั้งระบบเองได้ (รายละเอียดเต็มอยู่ใน M4-02)

# Web / API $ git clone <repo-url> && cd dis-api $ cp .env.example .env # กรอกค่าตามตาราง config (M4-02 ข้อ 3) $ <install deps> # ตามไฟล์ lock $ <run db migrate> $ <build> && <deploy> # Mobile (ต่อแอป) $ cd dis-mobile-patient $ <install> && <build android/ios> $ # ตั้ง API endpoint = Production แล้วส่ง App Store / Google Play
  • ระบุ prerequisite (เวอร์ชัน toolchain/SDK) ครบ
  • ทดสอบ build จากเครื่องสะอาดสำเร็จก่อนส่งมอบ
