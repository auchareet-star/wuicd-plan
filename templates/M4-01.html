3รายการตรวจสอบ Cutover (Cutover Checklist)

3.1 ก่อน Cutover (Pre-Cutover)

  • ยืนยันสภาพแวดล้อม Production พร้อม (server, network, certificate, license)
  • ยืนยันแผน Rollback และทดสอบขั้นตอนถอยแล้ว (ข้อ 5)
  • แจ้งผู้ใช้/คลินิกเรื่อง downtime และช่วง cutover แล้ว (ข้อ 7)
  • ทีม HIS เดิม ยืนยัน standby ตลอดหน้าต่างเวลา + ช่องทางติดต่อ
  • ตรวจ Go/No-Go criteria ครบทุกข้อ (ข้อ 4)

3.2 ระหว่าง Cutover

  • Backup — Full backup ฐานข้อมูล + ไฟล์ระบบเดิม สำเร็จและตรวจสอบ restore ได้
  • Freeze — ปิดรับ transaction ใหม่ในระบบเดิม + แจ้งหน่วยงานยุติการบันทึก
  • Migrate final — Migration delta รอบสุดท้ายเสร็จ ไม่มี error ค้าง
  • Reconciliation — เทียบยอดข้อมูลครบ 100% (จำนวนผู้ป่วย, นัดหมาย, การเงิน, ฟิล์ม PACS, ผล LIS)
  • Smoke test — ทดสอบ flow หลักผ่าน (ลงทะเบียน, จ่ายยา, ออกใบเสร็จ, เรียกฟิล์ม, ส่ง LIS)
  • Integration — เชื่อม HAIT Plus / SAP / PACS / LIS-RIS ทำงานถูกต้อง (ทดสอบ end-to-end)
  • เปิดให้ผู้ใช้ — ปลดล็อกระบบ DIS + ประกาศ Go-Live + แจ้งช่องทาง Hypercare

3.3 หลังเปิดใช้งานวันแรก

  • ตรวจ log/error วันแรก ไม่มี Critical/High ค้าง
  • ยืนยันธุรกรรมการเงินวันแรกตรงกับยอดจริง
  • สรุปประเด็นเข้า Hypercare (ข้อ 6) + ปิดประเด็นเร่งด่วน

5แผนถอยกลับ (Rollback Plan)

5.1 เงื่อนไขที่ต้องถอย (Rollback Trigger)

  • Reconciliation ไม่ครบ/ข้อมูลคลาดเคลื่อนแก้ไม่ได้ในเวลา
  • พบ defect ระดับ Critical ที่กระทบบริการผู้ป่วย
  • Integration หลัก (เช่น HAIT Plus / การเงิน SAP) ใช้งานไม่ได้

5.2 เวลาตัดสินใจถอย (Point of No Return) — ต้องตัดสินใจถอยภายใน __ ชั่วโมง หลัง migrate final และ ก่อน T-8 เปิดให้ผู้ใช้ · เวลาดำเนินการถอยให้เสร็จ (RTO) ภายใน __ ชั่วโมง

  • ยุติการใช้งาน DIS + แจ้งทุกหน่วยงาน
  • กู้คืนระบบเดิมจาก backup (T-3) และตรวจ restore ครบ
  • ปลด Freeze ระบบเดิม + เปิดให้บันทึกต่อ
  • บันทึกสาเหตุ + วางแผน cutover รอบใหม่

6การดูแลใกล้ชิดหลังขึ้นระบบ (Hypercare)

  • ระยะ Hypercare: __ วัน หลัง Go-Live มักกำหนด 14–30 วัน
  • ทีมประจำหน้างาน (on-site): SE __ คน, Dev __ คน, QA __ คน ครอบคลุมเวลาทำการคลินิก
  • ทีม HIS เดิม standby: ช่วยเหลือกรณีปัญหา Integration/ข้อมูลระบบเดิม ตลอดช่วง cutover และต้น Hypercare ตามที่จองคิวไว้
