3ตัวอย่างเนื้อหาแบบทีละขั้นตอน (Step-by-step)

3.1 การเปิดเวชระเบียนผู้ป่วย (เวชระเบียน / ทันตแพทย์)

  1. คลิกเมนู “เวชระเบียน” > “ค้นหาผู้ป่วย”
  2. กรอก HN หรือ เลขบัตรประชาชน หรือ ชื่อ–สกุล แล้วกด “ค้นหา”
  3. คลิกชื่อผู้ป่วยจากผลการค้นหาเพื่อเปิดเวชระเบียน
  4. ตรวจสอบข้อมูลพื้นฐาน ประวัติแพ้ยา และประวัติการรักษาก่อนหน้า

[ภาพประกอบ: __________________________ ]

3.2 การบันทึก Dental Chart (ทันตแพทย์ / นักศึกษา)

  1. จากเวชระเบียนผู้ป่วยที่เปิดอยู่ คลิกแท็บ “Dental Chart”
  2. เลือกซี่ฟันบนแผนภาพ (Tooth chart) ที่ต้องการบันทึก
  3. เลือก สภาพฟัน / หัตถการ จากรายการ (เช่น อุดฟัน, ถอนฟัน, รักษารากฟัน)
  4. ระบุพื้นผิว (Surface) และรายละเอียดเพิ่มเติม
  5. คลิก 💾 “บันทึก” — ระบบบันทึกเข้าประวัติการรักษาทันที
  6. นักศึกษา กด “ส่งให้อาจารย์อนุมัติ” ก่อนปิดเคส

[ภาพประกอบ: __________________________ ]

3.3 การจองนัด / Recall (ทันตแพทย์ / เวชระเบียน)

  1. จากเวชระเบียนผู้ป่วย คลิกเมนู “นัดหมาย” > “สร้างนัดใหม่”
  2. เลือก คลินิก/ทันตแพทย์ และ วันที่–เวลา ที่ว่าง
  3. เลือกประเภทนัด (ติดตามผล / Recall / ทำหัตถการต่อ)
  4. คลิก 💾 “บันทึกนัด” — ระบบส่งการแจ้งเตือนให้ผู้ป่วยตามช่องทางที่ตั้งไว้
